# LazyType website

Marketing site for [LazyType](https://github.com/SamNickGammer/lazytype.omprakashbharti.in) — fast AI dictation for macOS. Built by SamNickGammer.

## Stack

- [Next.js 14](https://nextjs.org/) (App Router)
- [Tailwind CSS 3](https://tailwindcss.com/)
- TypeScript

## Develop

```bash
cd website
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy

Deploys cleanly to [Vercel](https://vercel.com/) (set the project root to `website/`), or run `npm run build` and host the output yourself.
